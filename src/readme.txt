To run ranked insert
=-=-=-=-=-=-=-=-=-=-

'python run_rankedInsert.py -h' will give a list of arguments and options that are needed to run ranked insert. 

Example
*******

python run_rankedInsert.py -n 100 -m 1000 -k 10000 -t 1 -b 10

To run top-k

Example

python run_topk.py -n 100 -m 1000 -k 10 -t 1 -b 10
